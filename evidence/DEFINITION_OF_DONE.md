# Definition of Done

A backlog item is considered Done when:

- Backend endpoint is implemented.
- PostgreSQL persistence is implemented if needed.
- Server-side authorization is enforced.
- Frontend can demonstrate the behavior end to end.
- Error cases return clear messages.
- The feature is tested with demo data.
- Evidence screenshot or test note is prepared.
- Code is committed to GitHub.
- The task is moved to Done in ClickUp.
